# {% include icon.html icon="fa-solid fa-microscope" %}Research
我们的研究聚焦于城市形态、空间结构与城市演化的定量分析。我们利用先进的数据分析、仿真和建模技术，探索城市如何生长与变迁，旨在为可持续的城市发展与规划提供支持。

主要研究方向包括：
- 城市形态学与空间格局
- 城市增长建模与仿真
- 公共空间分析与优化
- 面向城市规划的数据驱动方法
